 Stage-2026-dipoles-codes
===

Ce dépôt contient tout les codes utilisés dans le cadre de mon stage de L3 à l'INPHYNI à Nice. Le stage s'intéressait à des méthodes d'études expérimentales et numériques de la collision 2D d'un dipôle tourbillonnaire.

# Dépendances 
 
Plus de précision plus tard mais matplotlib, numpy scipy et dedalus project probablement

# Utilisation 
 
## Codes expériences

### Calibrations

Ce dossier contient toutes les calibrations des appareils de mesures.

### Stratification

Ce dossier contient toutes les mesures datées des stratifications réalisées dans la cuve et le code matlab pour les calibrer.

### Vortex_Colorant

Ce dossier contient tout les résultats des observations qualitatives sur le comportement des dipôles lors de la collision dans différentes configurations.
